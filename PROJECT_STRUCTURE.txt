## 🏗️ Architecture Layers

```

                        ↓
```

- Install: "Desktop development with C++"
- Size: ~15 GB
- Time: 30-45 min
- ✅ Includes IDE

**Option 2: MinGW** (Lightweight)

- Install: x86_64 version
- Size: ~750 MB
- Time: 10-15 min
- ⚠️ Manual PATH setup needed

**CMake** (Build system)
- Download: https://cmake.org/download/
- Download: cmake-3.x.x-windows-x86_64.msi
- Size: ~50 MB
- Time: 5 min
- ✅ Check "Add to PATH" during install

## ⚡ Quick Setup (5 steps)

### 1️⃣ Install Visual Studio Community
```
Go to: https://visualstudio.microsoft.com/vs/community/
Download → Run → Select "Desktop development with C++" → Install
```

### 2️⃣ Install CMake
```
Go to: https://cmake.org/download/
Download: cmake-3.x.x-windows-x86_64.msi → Run → Check "Add to PATH" → Install
```

### 3️⃣ Verify Installation
```powershell
# Open PowerShell and run:
cl.exe              # Should show MSVC compiler info
cmake --version     # Should show version (e.g., 3.27.0)
```

### 4️⃣ Build the Project
```powershell
# Navigate to project folder:
cd c:\uet\coding\Cong_tac

# Run setup script:
.\setup.bat
# Follow menu prompts
```

### 5️⃣ Run the Application
```powershell
# After build completes:
.\build\Release\StudentManagement.exe
```

## 📞 Common Issues & Quick Fixes

**"cmake is not recognized"**
→ Uninstall & reinstall CMake, check "Add to PATH"

**"cl.exe is not recognized"**
→ Install Visual Studio or MinGW, close PowerShell & reopen

**Build fails**
→ Make sure compiler is installed before running CMake

**Can't find executable**
→ Check `build/Release/` (MSVC) or `build/` (MinGW) folder
