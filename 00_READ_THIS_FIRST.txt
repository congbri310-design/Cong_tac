## ⚡ 3-MINUTE SETUP SUMMARY

### STEP 1: Install Visual Studio
```
1. Go to: https://visualstudio.microsoft.com/vs/community/
2. Click Download
3. Run installer
4. Select: "Desktop development with C++"
5. Click Install (wait 30-45 min)
```

### STEP 2: Install CMake
```
1. Go to: https://cmake.org/download/
2. Download: cmake-3.x.x-windows-x86_64.msi
3. Run installer
4. ✓ Check "Add CMake to system PATH"
5. Complete (5 min)
```

### STEP 3: Verify Installation
```powershell
# Open PowerShell, type:
cl.exe              # Should show "Microsoft C/C++"
cmake --version     # Should show "cmake version 3.x"
```

### STEP 4: Build Project
```powershell
cd c:\uet\coding\Cong_tac
.\setup.bat
# Follow prompts, then run: .\build\Release\StudentManagement.exe
```

## 🆘 COMMON ISSUES & QUICK FIXES

| Issue | Solution |
|-------|----------|
| **"cmake is not recognized"** | Reinstall CMake, check "Add to PATH", restart PowerShell |
| **"cl.exe is not recognized"** | Install Visual Studio or MinGW, restart PowerShell |
| **Build fails** | Delete build folder, create new one, retry |
| **CMake can't find compiler** | Install compiler FIRST, then run CMake |
| **Don't know what to do** | Read [START_HERE.txt](START_HERE.txt) |
