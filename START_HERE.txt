## 📦 What You Need

### Step 1: Install Compiler (Choose ONE)

**Visual Studio Community** (Recommended)
- https://visualstudio.microsoft.com/vs/community/
- Select: "Desktop development with C++"
- Time: 30-45 min

**MinGW** (Lightweight)
- https://www.mingw-w64.org/
- Select: x86_64
- Time: 10-15 min

### Step 2: Install CMake

**CMake**
- https://cmake.org/download/
- Download: Windows x64 Installer
- ✓ Check "Add CMake to system PATH"
- Time: 5 min

### Step 3: Verify

```powershell
# Open PowerShell:
cl.exe              # Should show compiler info
cmake --version     # Should show version number
```

## 🚀 Build & Run

### Option 1: Using Batch Script (Easiest)
```powershell
cd c:\uet\coding\Cong_tac
.\setup.bat
# Follow menu prompts
# Then run: .\build\Release\StudentManagement.exe
```

### Option 2: Using CMake (Manual)
```powershell
cd c:\uet\coding\Cong_tac
mkdir build
cd build
cmake .. -G "Visual Studio 16 2019"
cmake --build . --config Release
# Then run: .\Release\StudentManagement.exe
```

## 🚨 Troubleshooting

### Can't find cmake
→ Add CMake bin folder to PATH, restart PowerShell

### Can't find compiler
→ Install Visual Studio or MinGW, restart PowerShell

### CMake can't find compiler
→ Make sure compiler installed FIRST, then use CMake

### Build fails
→ Delete build folder, create new one, try again

### Can't find executable
→ Check `build/Release/` (MSVC) or `build/` (MinGW)

More help in: [SETUP.md](SETUP.md) - Troubleshooting section
