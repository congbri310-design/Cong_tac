# 🚀 CÀI ĐẶT - HƯỚNG DẪN TIẾNG VIỆT

## ❌ VẤN ĐỀ

Máy bạn **THIẾU** 2 thứ để chạy dự án:
1. **Trình biên dịch C++** (Compiler)
2. **CMake** (Công cụ build)

---

## ✅ GIẢI PHÁP - CỰC KỲ ĐƠN GIẢN

### **CÁCH 1: Visual Studio Community (ĐỀ NGHỊ - Dễ nhất)**

#### Bước 1: Tải Visual Studio
```
1. Vào: https://visualstudio.microsoft.com/vs/community/
2. Bấm nút xanh "Download Visual Studio Community"
3. Tệp "VisualStudioSetup.exe" sẽ được tải về
4. Chạy file này
```

#### Bước 2: Cài đặt
```
1. Chờ installer mở ra
2. Tìm và chọn: "Desktop development with C++"
3. Bấm nút "Install" (góc dưới cùng bên phải)
4. Đợi khoảng 30-45 phút (phụ thuộc tốc độ internet)
5. Bấm "Launch" khi xong
```

**Kích thước**: 15 GB  
**Thời gian**: 30-45 phút  
**Ưu điểm**: Có IDE, dễ dùng, bao gồm mọi thứ cần thiết

---

### **CÁCH 2: MinGW (Nhẹ hơn)**

#### Bước 1: Tải MinGW
```
1. Vào: https://www.mingw-w64.org/
2. Tìm "Online Installer"
3. Tải file installer
4. Chạy file
5. Giữ mặc định, bấm "Next" đến khi xong
```

**Kích thước**: 750 MB  
**Thời gian**: 10-15 phút

---

### **BƯỚC 3: Cài CMake (Cần cho cả 2 cách)**

#### Hướng dẫn
```
1. Vào: https://cmake.org/download/
2. Tìm "Windows x64 Installer"
3. Tải file "cmake-3.x.x-windows-x86_64.msi"
4. Chạy installer
5. ⭐ QUAN TRỌNG: Chọn ✓ "Add CMake to system PATH"
6. Bấm "Install"
7. Chờ xong (~5 phút)
```

**⭐ NHỚ**: Bắt buộc phải chọn "Add CMake to system PATH"!

## 🆘 KHI NÓ LỖI

**Nếu nó nói "cmake is not recognized":**
- Bạn chưa chọn "Add to PATH" khi cài CMake
- Giải pháp: Gỡ CMake, cài lại, chọn "Add to PATH"

**Nếu nó nói "compiler not found":**
- Bạn chưa cài Visual Studio hoặc MinGW
- Giải pháp: Cài Visual Studio hoặc MinGW trước

**Nếu build fail:**
- Xóa folder "build"
- Chạy lại setup.bat
